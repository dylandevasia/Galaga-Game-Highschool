# Galaga Imitator

## Project Overview
**Galaga Imitator** is a Java-based arcade game inspired by the classic *Galaga*. Players control a spaceship, defeat enemies, and progress through increasingly challenging levels. This project was created as part of **Computer Science 3 coursework**.  

---

## Purpose
The purpose of this project is to demonstrate Java programming skills, game development concepts, and object-oriented design, while providing a fun and playable game experience.

## How to Run

Open a terminal in the project folder and run:

```bash
# Compile all Java files
javac -cp "lib/mayflower3.3.9.jar" *.java

# Run the game
java -cp "lib/mayflower3.3.9.jar:." GalagaRunner
